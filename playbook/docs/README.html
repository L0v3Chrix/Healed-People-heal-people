What this package does

Evidence in, roadmap out

This package turns the supplied Worth Keeping materials and completed interview into an evidence-based concept-to-beta and crowdfunding roadmap.

Gates, not assumptions

Recommendations are labeled, and legal, financial, partner, safety, privacy, print, and platform dependencies are handled as release gates rather than silent assumptions.
